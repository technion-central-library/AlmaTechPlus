Copy & Paste records in Metadata Editor

Alma >> Resources >> MDEditor

Step by step instructions

  1. Open MD Editor
  2. Click on the target Marc21 Bibliographic record that you want to replace. It will open on the left editor menu
  3. Click "Split Editor F6" in the top editor bar
  4. Click on the target Marc21 Bibliographic record that you want to duplicate, It will open on the right editor menu
  5. Click "Copy & Paste (right to left)" in the top editor bar
  6. Select "Reload Original Record" from the File menu to refresh the target record

Select and delete multiple fields in one click

Alma >> Resources >> MDEditor

Overview

This feature deletes all the selected fields of the recorded.

Step by step instructions
  1. Select the required fields using control (Ctrl) key. The select fields will marked in light blue
  2. Click the delete icon
  3. Confirm you want to delete by clicking the "OK" button on the right bottom
  4. The selected fields will be deleted
